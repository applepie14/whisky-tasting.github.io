Blind Tasting Reference

위스키 테이스팅 노트

Nose · Palate · Finish

👃
향 (Nose / Aroma)
잔을 코에 가까이 대고 첫 인상과 깊은 향을 탐색
과일 (Fruity)
사과배복숭아 살구자두건포도 무화과바나나오렌지 껍질 레몬파인애플체리
꽃 (Floral)
장미제비꽃재스민 라벤더헤더오렌지 꽃 풀꽃
곡물 / 견과 (Cereal / Nutty)
보리오트밀빵 아몬드헤이즐넛호두 마지판
달콤 (Sweet)
바닐라카라멜토피 꿀메이플 시럽버터스카치 초콜릿
나무 / 오크 (Woody)
오크삼나무송진 샌달우드오래된 나무연필심
스모키 / 피트 (Smoky)
피트모닥불 연기훈제 재타르병원약 아이오딘해조류
스파이시 (Spicy)
흑후추계피정향 생강넛맥아니스 고수
와인 / 주류 (Winey)
셰리포트레드 와인 건자두다크 베리포도
유제품 / 크리미 (Creamy)
버터크림우유 치즈사워크림
미네랄 / 흙 (Earthy)
습한 흙이끼버섯 젖은 돌가죽담배
👅
맛 (Palate / Taste)
입안에 머금고 혀의 각 부위에서 느껴지는 풍미 분석
단맛 (Sweetness)
꿀갈색 설탕바닐라 크림 카라멜잼말린 과일
과일 (Fruit)
그린 애플시트러스복숭아 망고무화과베리류
스파이시 (Spice)
화이트 페퍼계피생강 칠리 열감정향올스파이스
오크 / 나무 (Oak)
토스티 오크타닌시더 바닐린코코넛
구운 것 (Roasted)
다크 초콜릿에스프레소캐러멜라이즈 구운 곡물토스트
스모키 / 피트 (Smoke)
BBQ피트 흙연기 크레오소트짭짤한 재
짠맛 / 미네랄 (Saline)
바닷소금올리브굴 미네랄금속성
쓴맛 (Bitter)
다크 초콜릿 쓴맛에스프레소 호피타닌 드라이
산미 (Acidic)
레몬그린 애플 산발사믹 요거트와인 산미
텍스처 (Texture)
오일리실크크리미 워터리두꺼운떫음
